# 🌐 Personal Portfolio Website

Hi! I'm **Preeth Nair** 

This repository contains the code for my **personal portfolio website**, where I showcase my work, projects, and skills related to **Data Science, Machine Learning, and Artificial Intelligence**.

I built this website to create a simple place where recruiters, collaborators, or anyone interested can quickly explore the kind of work I do.

---

## 🚀 What this website includes

* 👨‍💻 **About Me** – A quick introduction about my background and interests
* 📊 **Projects** – Some of the data science and machine learning projects I've worked on
* 🧠 **Skills** – Tools and technologies I use in my work
* 📄 **Resume** – Easy access to my professional resume
* ✉️ **Contact** – A simple way to reach out to me

The site also includes some interactive elements like **smooth scrolling, particle background animation, and subtle UI effects** to make the experience more engaging.

---

## 🛠 Technologies Used

This portfolio was built using:

* **HTML**
* **CSS**
* **JavaScript**

No heavy frameworks — just clean and simple web technologies.

---

## 🎯 Why I Built This

As someone learning and working in **Data Science and Machine Learning**, I wanted a dedicated place to present my projects and experiments in a structured way.

This portfolio helps me:

* showcase my work
* track my progress
* share projects with recruiters and the tech community
